# Sweet Crust Bakery

A simple, beginner-friendly static website for a family-run bakery. Built with plain HTML, CSS and JavaScript — no build tools required.

## Features

- Hero section with tagline and "Order Now" button
- Menu with three featured items
- About and Contact sections
- Responsive layout with warm bakery colours (cream, brown, soft pink)

## Getting started

1. Clone or download this repository.
2. Open `index.html` in your favourite browser — just double-click the file, or right-click and choose _Open With_.

That's it! No installation, no dependencies.
